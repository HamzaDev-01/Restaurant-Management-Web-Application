Testing Cart Functionality

If you see this page, the server is running correctly.

To see the cart notification changes:

  1. Press Ctrl + Shift + Delete to open browser cache settings
  2. Clear all cache and cookies
  3. Or press Ctrl + F5 for a hard refresh
  4. Go back to http://localhost:8000
  5. Try adding an item to cart - you should see a green notification!
